## 本demo可以了解学习到如下知识：
- 从头开始创建一个Flutter应用程序.
- 编写 Dart 代码.
- 利用外部的第三方库.
- 使用热重载加快开发周期.
- 实现一个有状态的widget，为你的应用增加交互.
- 用ListView和ListTiles创建一个延迟加载的无限滚动列表.
- 创建了一个路由并添加了在主路由和新路由之间跳转逻辑
- 了解如何使用主题更改应用UI的外观.


## 第三方
- [intl](https://pub.dev/packages/intl) 时间格式化
- [fish_redux](https://pub.dev/packages/fish_redux) 状态数据流管理
